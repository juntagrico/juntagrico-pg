Welcome to the juntagrico-pg documentation
==========================================================
A postgres editor for juntagrico. It allows you to run queries form juntagrico. Users need special permission to see the view.

juntagrico-pg is an extension for juntagrico.

Head over to 
`juntagrico.readthedocs.io <https://juntagrico.readthedocs.io/en/latest/>`_ for information on how to setup juntagrico in order to be able to use this app.
